# Cube Fighter

## Overview

Cube Fighter is a fast-paced, survival-style arcade game, developed in Unity. The game focuses on defeating increasingly difficult waves of enemies while surviving as long as possible in a minimal, cube-based arena.

This project was built to explore enemy spawning systems, player progression, and real-time combat mechanics using custom C# scripts and original game design.

## Features

### Survival-Style Gameplay

- Continuous enemy waves that scale in difficulty over time

- Score and survival-time focused gameplay loop

### Custom Enemy Systems

- Dynamic enemy spawning and scaling

- Multiple enemy behaviors and movement patterns

- Wave and timing control through C# scripts

### Player Combat & Progression

- Player movement and attack mechanics

- Increasing difficulty based on survival time

- Real-time collision detection and damage handling

### Original Game Design

- Custom arena layout and gameplay tuning

- Cube-based visual style for clarity and performance

- Fully original mechanics and implementation

## Tech Stack

- Engine: Unity

- Language: C#

- Scripting: Custom MonoBehaviour scripts

- Version Control: Git / GitHub

## How to Edit or Run
1. Clone the repository:
```
git clone https://github.com/your-username/cube-fighter.git
```

2. Open Unity Hub

2. Click Open Project and select the cloned project folder

4. Open the main scene

5. Customize game or press play in the Unity Editor
